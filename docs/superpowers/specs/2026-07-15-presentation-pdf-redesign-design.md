# 발표용 수요조사 PDF 리디자인

## 목적

웹 리서치 페이지와 전혀 다른 구형 보고서 양식으로 생성되는 PDF를 교체한다. 새 PDF는 웹페이지의 조사 서사와 디자인 언어를 유지하면서 발표자가 바로 공유할 수 있는 4페이지 핵심 보고서로 제공한다.

## 성공 기준

- 다운로드 시점의 한국인·외국인 설문 집계를 사용한다.
- 웹페이지와 같은 네이비, 청록, 코랄, 흰색 중심의 시각 체계를 사용한다.
- 조사 결과가 `문제 발견 -> 시냅스팟 기능 연결` 순서로 읽힌다.
- 각 페이지에 집단명, 표본 수, 생성 시각과 페이지 번호가 명확하게 표시된다.
- 한글이 깨지거나 텍스트·그래프·이미지가 겹치지 않는다.
- Google Sheets가 갱신되면 별도 수정 없이 새 수치가 PDF에 반영된다.

## 범위

### 포함

- 기존 `현재 데이터로 PDF 내려받기` 버튼 유지
- ReportLab 기반 PDF 생성 코드 재설계
- 시냅스팟 실제 모바일 미리보기 이미지 2개 포함
- 조사 핵심 지표, 문제별 막대그래프, 제품 연결 설명 포함
- PDF 데이터와 화면 데이터의 표본 수 일치 검증
- 4페이지 렌더링 및 PNG 시각 검증

### 제외

- 웹페이지 전체 상세 결과와 익명 자유 의견을 PDF에 모두 복제
- 브라우저 화면 캡처 또는 인쇄 대화상자 자동화
- Synapspot 저장소의 UI 변경
- Google Form 또는 Google Sheets 구조 변경

## 페이지 구성

### 1페이지: 조사 개요와 핵심 결과

- 제목: `교류학생의 제주 생활, 무엇이 가장 불편했을까?`
- 한국인·외국인 조사 목적 설명
- 전체 응답, 한국인 응답, 외국인 응답, 서비스 긍정 의향 카드
- 한국인 핵심 불편과 외국인 핵심 불편을 짧은 결론으로 표시
- 데이터 출처와 PDF 생성 시각 표시

### 2페이지: 문제 1 - 이동과 동행 모집

- 한국인 설문의 제주 생활 불편 상위 항목
- 한국인 설문의 오픈채팅 수요 상위 항목
- 택시팟과 버스 노선 수치를 강조
- `같은 시간과 목적지를 가진 사람을 빠르게 찾는 문제`라는 해석 표시

### 3페이지: 문제 2 - 공지와 생활정보 탐색

- 외국인 설문의 제주 생활 불편 상위 항목
- 외국인 설문의 오픈채팅 수요 상위 항목
- 교통과 공지 수치를 강조
- `정보를 한곳에서 찾고 신뢰도를 확인하는 문제`라는 해석 표시

### 4페이지: 조사 결과에서 제품으로

- `이동·동행 파티`와 `근거 있는 AI 질문` 기능 연결
- 실제 시냅스팟 모바일 화면 2개
- 설문은 기능 효과를 입증한 것이 아니라 제품 방향을 정한 근거라는 주의 문구
- 공개 리서치 페이지 주소 표시

## 구현 방식

`build_current_pdf()`는 현재 화면에서 사용하는 두 설문 객체를 그대로 입력받는다. 별도의 하드코딩된 응답 수를 사용하지 않고 `build_research_view_model()`과 동일한 원천 데이터에서 핵심 수치를 계산한다.

PDF 전용 그리기 함수는 다음 책임으로 나눈다.

- 공통 페이지 배경, 헤더, 푸터, 페이지 번호
- 지표 카드
- 막대그래프와 강조 수치
- 설명문과 주의 문구
- 모바일 미리보기 이미지 배치

ReportLab을 유지해 Streamlit Cloud에서 추가 브라우저나 시스템 렌더러 없이 생성되도록 한다. 완성된 PDF는 `pdftoppm`으로 PNG 렌더링하여 모든 페이지를 시각 검증한다.

## 오류 처리

- 미리보기 이미지가 없으면 PDF 생성을 실패시키지 않고 안내 영역으로 대체한다.
- 응답 항목이 비어 있으면 `표시할 데이터가 없습니다`를 표시한다.
- 한글 TTF가 없으면 기존 CID 폰트 대체 경로를 유지한다.
- PDF 생성에는 화면과 동일하게 Google Sheets 실패 시 CSV 대체 데이터를 사용한다.

## 테스트

- PDF가 정확히 4페이지인지 검사한다.
- PDF 텍스트에 전체·한국인·외국인 표본 수가 포함되는지 검사한다.
- 두 문제 제목과 시냅스팟 기능명이 포함되는지 검사한다.
- 미리보기 이미지가 없어도 PDF 생성이 성공하는지 검사한다.
- 생성 PDF를 PNG로 렌더링하여 잘림, 겹침, 한글 깨짐, 이미지 비율을 확인한다.
